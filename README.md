# testTau


1. change the pythia8 installation in the env.sh file, 
i.e. replace this line:

```
export PYTHIA8=/Users/lihengne/local/llp/pythia8313
```
to point to your own pythia8 installation


2. setup the env:

```
source env.sh
```

3. compile the codes

```
source compile.sh
```

4. run it

```
./main42
```


5. drow plots

```
pythin -i plot.py 
```
or
```
python -i plot2.py
```
